Tutorials
==========
This page contains a list of tutorials that are available to help the users get started with STEM.
The tutorials cover a range of topics, from basic usage to more advanced features.
Each tutorial is designed to be self-contained.
However it is recommended to follow them in order, as the tutorials build on functionalities and concepts
introduced in the previous ones.

Benchmark report
================
The STEM benchmark report contains a comprehensive set of benchmarks that are performed to validate and benchmark the
STEM model against analytical solutions. This report is automatically generated.

The report for version 1.3 is available at: `STEM benchmark report v1.3 <https://github.com/StemVibrations/STEM/releases/download/v1.3.0/benchmark_report_v1_3_0.pdf>`_.
The benchmark report for the current development version of STEM is available at: `STEM benchmark report latest <https://github.com/StemVibrations/STEM/releases/download/pdf-latest/benchmark_report.pdf>`_.
